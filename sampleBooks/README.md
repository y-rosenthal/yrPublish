# What's in this folder

This folder contains folders which contain sample books.

## All folders ... except for the otherBooks folder

Each subfolder (other than the otherBooks folder) contains possibly
different versions of the same sample book.
Within the folder of a sample book could be several folders with
the naming format 

   nameOfSampleBook-vn.n.n
   
where each n.n.n is the version number of the yrPublish system
that was used to create the sampleBook in that folder.

## special v0.0.0 version number

The "special" version number v0.0.0 represents a book
that was created prior to yrPublish existing.
Those books serve as the inspiration of yrPublish v0.0.1.

## otherBooks folder

The otherBooks folder contains experimental books. Most of these books
were create as experiments when working on designing yrPublish. 
In general, you should not look at these experimental books as examples
unless explicitly told to do so.
